# AInTandem

一個致力於實現**人與 AI 共生創作場**的開源專案。

## 專案願景

AInTandem 的終極目標，是成為一個能夠八成獨立運作的軟體工程團隊。

不是輔助工具，不只是 AI coding，而是一群可以被長時間交付任務、能自己在環境裡協作、產出可用成果的 AI agents。

當你只有一個人，但你需要一個工程團隊的時候，AInTandem 可以補上那八成。

## 內容分類

本站部落格文章依據以下主題分類：

### 全部文章
所有文章的總覽，按發布時間排序。

### 公告 (Announcement)
專案的重要公告、版本發布、以及重大更新訊息。

### 架構與技術 (Architecture)
系統架構設計、技術決策、實作細節的深度探討。包含：
- Sandbox / Workspace 管理機制
- Network routing 與 port forwarding
- 基礎網路與隔離設計
- Agent 協作模式
- 技術債與重構紀錄

### 思想與隨筆 (Thoughts)
關於 AI agent 發展的思考與觀察。包含：
- 對 AI 協作模式的理解
- 從「沙盒」到「Coworking Space」的概念演進
- YOLO mode：讓 AI 在安全環境中自主行動
- 長時間任務交付的信任問題

### 路線圖 (Roadmap)
專案發展規劃與進度追蹤。包含：
- 版本里程碑
- 功能發展藍圖
- 已完成與進行中的工作項目
- 未來願景與預期時程

### 專案理念 (Philosophy)
AInTandem 的核心價值與設計哲學。包含：
- 為什麼不讓 AI 取代人，而是與人並行協作
- 如何建立「心理上能放得下來」的基礎建設
- 從概念到可持續運作的系統骨架
- 開源與透明的開發過程
